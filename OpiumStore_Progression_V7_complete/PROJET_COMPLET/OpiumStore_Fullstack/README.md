# Opium Store Hub — Render + Cloudflare + Discord

Projet prêt à déployer avec :

- frontend statique Render sans veille ;
- connexion Discord OAuth2 ;
- sessions serveur ;
- base Cloudflare D1 ;
- générateur FIFO avec cooldown ;
- boutique à points FIFO ;
- Wallet par utilisateur ;
- roue de points ;
- panel admin par Discord ID ;
- chiffrement AES-GCM des lignes stockées.

Commence par lire `DEPLOIEMENT.md`.
